## Platzhalter

:::info Preview Feature nicht überall verfügbar
:::

:::note Bitte exakte Schreibweise (Groß-/Kleinschreibung) beachten
:::

Einzelne Dokumente können als Vorlagen erstellt werden. Diese Vorlagen können genutzt werden, um automatische Reports zu erstellen.
Um das Ausfüllen mit Daten aus Ihrer Software zu ermöglichen, muss das Schlagwort <code>\_dynamicContent</code> verknüpft werden. Platzhalter werden gefunden, in dem die Variable in <code>{{variable}}</code> in doppelt geschweiften Klammern verwendet wird.

Folgende Platzhalter können verwendet und ersetzt werden:

| Platzhalter                        | Ursprung                                     |
| ---------------------------------- | -------------------------------------------- |
| organisation                       | Firma aus Admin Startseite                   |
| {{organisation.title}}             | Name der Firma                               |
| managingDirector                   | Geschäftsführer aus Admin Startseite         |
| {{managingDirector.fullName}}      | Vor- und Nachname GF                         |
| {{today}}                          | Aktuelles Datum                              |
| dataProtectionOfficer              | Datenschutzbeauftragter aus Admin Startseite |
| {{dataProtectionOfficer.fullName}} | Vor- und Nachname DSB                        |
